# Odin Landing Page

Proyek *landing page* sederhana yang dibuat sebagai bagian dari kurikulum **The Odin Project**. Fokus utama proyek ini adalah mengasah kemampuan **Flexbox** dan struktur **HTML/CSS**.

## 🛠️ Build With
* **HTML5**
* **CSS3** (Custom Properties & Flexbox)

## 🚀 Live Preview
Kamu bisa melihat hasilnya di sini:  https://asyahi-helmy.github.io/landing-page-odin/

## 📝 Catatan Proyek
Tujuan dari proyek ini adalah:
* Memahami penyelarasan elemen dengan Flexbox.
* Mengatur tata letak visual berdasarkan desain mockup.
* Menerapkan struktur HTML yang semantik.
